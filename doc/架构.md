# Agent智能评测平台 — 前端架构文档

## 一、项目概述

Agent智能评测平台是一个面向 AI Agent 评估与基准测试的企业级单页应用，支持多 Agent 管理、评测任务调度、数据集管理、对比分析和报告生成。

## 二、技术栈

| 分类 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 框架 | Vue 3 | ^3.5 | Composition API + `<script setup>` |
| 语言 | TypeScript | ~5.7 | 严格模式 |
| 构建 | Vite | ^6.0 | 路径别名、分包、代理 |
| 路由 | Vue Router | ^4.5 | 懒加载、导航守卫 |
| 状态管理 | Pinia | ^2.3 | 模块化 stores |
| HTTP | Axios | ^1.7 | 拦截器、统一错误处理 |
| 样式 | Tailwind CSS | ^3.4 | 自定义 AI-Native 主题 |
| 图表 | ECharts | ^5.5 | 深色主题 |
| 图标 | Lucide Vue Next | ^0.468 | SVG 图标集 |
| 工具库 | @vueuse/core | ^12.3 | 组合式工具 |
| 时间 | dayjs | ^1.11 | 轻量日期处理 |

## 四、路由设计

| 路径 | 页面 | 说明 |
|------|------|------|
| `/` | — | 重定向至 `/dashboard` |
| `/dashboard` | DashboardPage | 仪表盘主页 |
| `/agents` | AgentListPage | Agent 列表 |
| `/agents/:id` | AgentDetailPage | Agent 详情 |
| `/evaluation` | EvaluationPage | 评测任务列表 |
| `/evaluation/:id` | EvaluationDetailPage | 任务详情 |
| `/comparison` | ComparisonPage | 对比分析（雷达图） |
| `/datasets` | DatasetPage | 数据集管理 |
| `/reports` | ReportPage | 评测报告 |
| `/system` | SystemPage | 系统设置 |
| `/:pathMatch(.*)*` | NotFoundPage | 404 |

路由使用懒加载 (`() => import(...)`) 并设置 `meta.title` 用于文档标题更新。

## 五、状态管理 (Pinia)

| Store | 职责 |
|-------|------|
| `useAppStore` | 全局 UI：侧边栏折叠、主题切换 |
| `useAgentStore` | Agent CRUD 列表、分页、筛选 |
| `useEvaluationStore` | 评测任务、结果、数据集 |

## 六、API 层设计

```
请求 → client.ts (Axios实例)
     ├── 请求拦截：注入 Authorization Token
     ├── 响应拦截：统一数据解包 (response.data)
     └── 错误拦截：401 清除 Token 跳转登录
     ↓
modules/ (业务 API 封装)
     ├── agentApi.list / detail / create / update / remove / evaluate
     ├── dashboardApi.stats / trend
     └── evaluationApi.listTasks / createTask / results / compare / listDatasets
```
